Roo Code

Community • null
Roo Code (formerly Roo Cline) is an AI-powered autonomous coding agent that lives in your editor. It's a fork of Cline with additional experimental features, offering comprehensive file management, command execution, browser automation, and integration with multiple AI models. It provides features for creating and running entire workflows using prompts to large language models, with capabilities like intelligent context condensing, fast edits, and orchestration with large context windows.
