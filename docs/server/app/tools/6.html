Zed

EDITOR

Company: Zed Industries

Legal Name: Zed Industries Inc.

Zed is a high-performance, open-source, multiplayer code editor designed for collaboration and AI-assisted coding. Written from scratch in Rust, it efficiently leverages multiple CPU cores and GPU for fast performance, offering modern features like built-in debugger capabilities, syntax-aware editing, and agentic editing mode. It supports multiple programming languages and integrates various AI models for code assistance.
